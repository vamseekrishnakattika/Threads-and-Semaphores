The entire code is in the file Project2.c

Compile instructions:

To compile the file, use the command gcc followed by the filename and the command -lpthread. a.out file is generated with the following command.

Ex: gcc Project2.c -lpthread

If we want to name our output file other than a.out then we have to use the following command.

gcc Project2.c -o project2 -lpthread.

Using the above command the object file generated is project2

Execute instrcutions:

To execute the file, use the following command

Ex: ./a.out or ./project2
